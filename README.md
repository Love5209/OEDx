# OEDx 博客

## 贡献方法

1. 安装 hexo ：

``` sh
npm install -g hexo-cli
```

如果 npm 过慢，可以选择使用 tnpm 。

2. fork 本仓库到你的 Github 账户下
3. 克隆你的仓库

``` sh
git clone https://github.com/你的Github账户/OEDx.github.io.git
```

4. 写文章，同时可以在本地（http://localhost:4000）开启预览：

``` sh
hexo server
```

5. 提交你的改动
6. 发 Pull Request 等待管理员审核
